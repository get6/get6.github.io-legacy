Naver Pay API 가이드

March 3월 13일, 2020

Naver Pay 가이드에서 GET STARTED 결제창 호출을 설명해주는 화면이 있습니다.

client-id, merchant-user-key, merchant-pay-key을 가져올 수 있는 방법이 없어보입니다.

이 세가지가 없는 경우 올바른 실행을 할 수 없어 관련 값을 얻어올 수 있는 곳을 찾아봤지만 찾을 순 없었습니다.

Naver Pay Center에서 등록되지 않은 사업자는 사용할 수 없어보입니다.
