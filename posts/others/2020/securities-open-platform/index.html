증권사 API 정리

March 11, 2020

다수의 증권사들이 오픈플랫폼이라는 명칭으로 open api를 제공하고 있습니다.

각 증권사들이 제공하는 API가 있으면서 한국예탁결제원(SEIBro), 한국거래소 지분을 76%가지고 있는 Koscom이라는 회사도 API를 제공합니다.

  • koscom 개발자 센터에서API 이용신청을 하실 수 있습니다.
  • SEIBro 오픈플랫폼에서 API 활용신청을 하실 수 있습니다.
  • 키움증권 Open API+에서 사용 신청하러 가기를 선택합니다.
  • eBEST투자증권 xingAPI을 최신버전 설치합니다. 설치 가이드를 알려주는 사이트도 있습니다 - API가 파이썬으로 작성되었습니다.
  • NH투자증권은 SPC로 되어있어 URL이 변경되지 않습니다. 트레이딩채널 / QV Open API로 들어갑니다. - C++를 지원합니다.

각 open API 제공 사이트 마다 개발자 회원가입 시 소속회사 사업자번호와 대표자명을 요구하므로 미리 알아두는게 좋습니다.
